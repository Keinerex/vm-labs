# ЛР №2 — Численные методы решения СЛАУ. Вариант 0

## Система уравнений

```
(1+a)x₁ + 14x₂  − 15x₃  + 23x₄ = 5
  16x₁  + (1+b)x₂ − 22x₃ + 29x₄ = 8
  18x₁  +  20x₂  − (1+c)x₃ + 32x₄ = 9
  10x₁  +  12x₂  −  16x₃ + (1+d)x₄ = 4
```

a, b, c, d — последние 4 цифры зачётной книжки.

---

## Метод 1 — Метод Крамера

**Идея:** решение находится через отношение определителей.

$$x_i = \frac{\det(A_i)}{\det(A)}$$

где $A_i$ — матрица $A$, в которой $i$-й столбец заменён на вектор $b$.

**Условие существования:** $\det(A) \neq 0$.

**Алгоритм:**
1. Вычислить $\det(A)$.
2. Для каждого $i$ заменить $i$-й столбец на $b$, вычислить $\det(A_i)$.
3. $x_i = \det(A_i) / \det(A)$.

**Трудоёмкость:** $O((n+1)!)$ — неэффективен уже при $n > 4$.

---

## Метод 2 — Метод прогонки (метод Гаусса)

**Идея:** последовательным исключением неизвестных привести систему к верхнетреугольному виду, затем найти решение подстановкой.

### Прямой ход

Для каждого шага $k = 1, \ldots, n$:

$$a_{ij}^{(k)} = a_{ij}^{(k-1)} - \frac{a_{ik}^{(k-1)}}{a_{kk}^{(k-1)}} \cdot a_{kj}^{(k-1)}, \quad i > k$$

Результат — верхнетреугольная матрица:

```
[α₁₁  α₁₂  α₁₃  α₁₄ | β₁]
[ 0   α₂₂  α₂₃  α₂₄ | β₂]
[ 0    0   α₃₃  α₃₄ | β₃]
[ 0    0    0   α₄₄ | β₄]
```

### Обратный ход

Начиная с последнего уравнения:

$$x_n = \frac{\beta_n}{\alpha_{nn}}, \qquad x_i = \frac{1}{\alpha_{ii}}\!\left(\beta_i - \sum_{j=i+1}^{n} \alpha_{ij}\, x_j\right)$$

**Определитель** вычисляется как произведение диагональных элементов после прямого хода: $\det(A) = \alpha_{11} \cdot \alpha_{22} \cdots \alpha_{nn}$.

**Трудоёмкость:** $O(n^3/3)$.

---

## Метод 3 — Метод простой итерации

**Идея:** разрешить $i$-е уравнение относительно $x_i$ и итерировать.

### Построение итерационной схемы

$$x_i^{(k+1)} = \beta_i + \sum_{j=1}^{n} \alpha_{ij}\, x_j^{(k)}$$

где:

$$\beta_i = \frac{b_i}{a_{ii}}, \qquad \alpha_{ij} = \begin{cases} -\dfrac{a_{ij}}{a_{ii}}, & j \neq i \\ 0, & j = i \end{cases}$$

Начальное приближение: $x_i^{(0)} = \beta_i$.

### Условие сходимости

Процесс сходится, если **хотя бы одна** из норм матрицы $\alpha$ меньше 1:

| Норма | Формула | Название |
|-------|---------|----------|
| $\|\alpha\|_m$ | $\max_i \sum_j \|\alpha_{ij}\|$ | Строчная (∞-норма) |
| $\|\alpha\|_l$ | $\max_j \sum_i \|\alpha_{ij}\|$ | Столбцовая (1-норма) |
| $\|\alpha\|_k$ | $\sqrt{\sum_{ij} \alpha_{ij}^2}$ | Евклидова (F-норма) |

**Достаточное условие (диагональное преобладание):**

$$|a_{ii}| > \sum_{j \neq i} |a_{ij}|, \quad \forall\, i$$

### Условие остановки

$$\|x^{(k+1)} - x^{(k)}\|_\infty < \varepsilon, \quad \varepsilon = 10^{-5}$$

## Вопросы самоконтроля (краткие ответы)

**Прямые методы** дают точное решение за конечное число операций. Итерационные — приближают решение последовательными приближениями.

**Метод Гаусса устойчив**, если матрица имеет диагональное преобладание: $|a_{ii}| > \sum_{j\neq i}|a_{ij}|$.

**Число обусловленности** $\text{cond}(A) = \|A\| \cdot \|A^{-1}\|$. Если $\text{cond}(A) \sim 10^k$, $k > 2$ — матрица плохо обусловлена, малые ошибки в $b$ дают большие погрешности в $x$.

**Невязка** $r = b - Ax$ — критерий качества решения. Малая невязка не всегда означает точное решение (при плохой обусловленности).

**Итерации расходятся** для данной системы, т.к. диагональные элементы (≈1) много меньше суммы внедиагональных. Для достижения сходимости нужно переупорядочить уравнения так, чтобы наибольший элемент каждой строки оказался на диагонали.
